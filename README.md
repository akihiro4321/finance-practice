# Finance Practice

新任マネージャー向け会計学習アプリ

[![Development Status](https://img.shields.io/badge/Phase-1%20Complete-success)](docs/development/phase1-completion.md)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

## 🎯 概要

新任マネージャーのための会計知識習得を目的としたプラクティス型学習アプリケーションです。特にシステム開発費用の会計処理と財務3表の理解に焦点を当てています。

### 対象ユーザー
- 新任マネージャー、プロジェクトリーダー
- 基本的な会計用語の理解がある方
- 研修・学習目的での利用

## ✨ 現在の機能（全7フェーズ完成 🎉）

### 1. システム開発費用シミュレータ
- 開発フェーズの選択（要件定義・開発・運用保守）
- 費用化・資産化の選択機能
- 判断基準に基づいたガイダンス
- 会計仕訳の自動生成
- 減価償却見込み計算

### 2. 財務三表ダッシュボード
- **損益計算書（P/L）**: 売上、費用、利益の分析
- **貸借対照表（B/S）**: 資産、負債、純資産の状況
- **キャッシュフロー計算書（C/F）**: 現金の流れの把握
- **シナリオ比較機能**: 複数の条件での財務影響分析

### 3. プロジェクト予算策定ツール
- **予算項目管理**: カテゴリ別の詳細予算作成
- **ROI/IRR/NPV計算**: 投資効果の定量分析
- **リスク分析**: 予算リスクの自動評価
- **改善提案**: AI による予算最適化提案

### 4. 実践学習モジュール
- **3つのケーススタディ**: 
  - ECサイトリニューアルプロジェクト
  - クラウド移行プロジェクト  
  - AI システム開発プロジェクト
- **5つの実践演習**: ROI計算、減価償却、予算分析など
- **進捗ダッシュボード**: 学習状況の可視化と分析
- **パーソナライズ推奨**: 弱点に基づく学習プラン提案

## 🚀 クイックスタート

### 前提条件
- Node.js 16.0.0 以上
- npm または yarn

### インストールと起動

```bash
# リポジトリのクローン
git clone https://github.com/akihiro4321/finance-practice.git
cd finance-practice

# 依存関係のインストール
npm install

# 開発サーバーの起動
npm start
```

アプリケーションは http://localhost:3000 で利用可能になります。

### ビルド

```bash
# プロダクション用ビルド
npm run build
```

## プロジェクト構成

```
src/
├── components/           # React コンポーネント
│   ├── Layout/          # レイアウト関連（Header, Navigation）
│   ├── Simulator/       # システム開発シミュレーター
│   ├── FinancialStatements/ # 財務三表ダッシュボード
│   ├── Budget/          # 予算策定・ROI分析
│   ├── Learning/        # 実践学習モジュール
│   └── ProjectForm/     # プロジェクト入力フォーム
├── types/               # TypeScript 型定義
│   ├── index.ts         # 基本型定義
│   ├── budget.ts        # 予算関連型
│   └── learning.ts      # 学習関連型
├── utils/               # ユーティリティ関数・計算エンジン
│   ├── accounting.ts    # 会計計算ロジック
│   ├── financial.ts     # 財務分析ロジック
│   ├── budgetCalculations.ts # 予算・ROI計算
│   └── validation.ts    # バリデーション
├── data/                # 静的データ
│   ├── caseStudies.ts   # ケーススタディデータ
│   └── exercises.ts     # 演習問題データ
├── styles/              # CSS ファイル
└── docs/                # 設計ドキュメント
    ├── specs/           # 要件・仕様書
    ├── design/          # アーキテクチャ・設計書
    └── development/     # 実装・開発ガイド
```

## 🏗️ システムアーキテクチャ

本アプリケーションは4つの主要モジュールで構成されています：

1. **🔧 システム開発シミュレータ**: 会計処理の判断と仕訳生成
2. **📊 財務三表ダッシュボード**: P/L・B/S・C/Fの分析・比較
3. **💰 予算策定ツール**: ROI分析とリスク評価  
4. **📚 実践学習モジュール**: ケーススタディと進捗管理

詳細なアーキテクチャ図とシーケンス図は[設計書](docs/design/)をご参照ください。
